# Specification

## Summary
**Goal:** Build AXOPAY Digital Wallet Platform with investment plans, referral rewards, binary matching system, and comprehensive admin controls.

**Planned changes:**
- Create public landing page with hero section, about, problem-solution, security, global capability, plans, referral system, and CTA sections
- Implement client authentication (login, sign-up with OTP verification, forgot password)
- Implement admin authentication portal with 2FA and optional IP restriction
- Create client dashboard displaying wallet balances (Main, Earnings, Withdrawal), active plan, ROI progress, earnings breakdown, team metrics, and recent transactions
- Implement wallet features: deposit, withdraw, internal transfer, currency conversion, and transaction history
- Implement investment plan management: purchase/upgrade plans (BEGIN $100, GROW $300, PEAK $500), ROI tracker, expiry view, and repurchase
- Implement automated ROI engine crediting 5% daily returns for 30 days to Earnings wallet
- Implement referral module with personal referral link, downline list, team tree visualization, and 10% direct bonus tracking
- Implement binary system with left/right team structure, pair matching logic, daily limits, and volume carry-forward
- Implement binary matching engine crediting $10/$30/$50 per pair based on plan tier
- Implement milestone rewards with progress indicators and claimed rewards history
- Implement earnings and transactions section with filters by date, type, and status
- Implement profile and settings: personal info update, password change, security settings, optional KYC upload, and notification preferences
- Create admin panel overview dashboard with key metrics: total users, active users, new registrations, deposits, withdrawals, revenue, and pending approvals
- Implement admin user management: view details, activate/suspend accounts, reset passwords, adjust balances, modify plans, and view referral trees
- Implement admin financial control: approve/reject deposits and withdrawals with manual entry and processing tracking
- Implement admin plan management: create/edit plans, set ROI percentage and duration, enable/disable availability
- Implement admin referral and binary configuration: set referral percentage, pair rules, daily limits, and carry-forward logic
- Implement admin rewards management: define milestones, configure reward amounts, and track payouts
- Implement admin reports and analytics: revenue reports, user growth charts, earnings distribution, transaction logs, and CSV export
- Implement admin notifications: system announcements, email campaigns, and platform alerts
- Implement backend wallet engine handling three wallet types, internal transfers, transaction fees, and currency conversion
- Implement security features: password hashing, withdrawal verification, fraud monitoring, admin activity logs, rate limiting, and backup systems
- Apply cohesive visual design with professional financial aesthetic using distinctive color palette (non-blue/purple), consistent typography, and trust-focused layout

**User-visible outcome:** Users can register with referral IDs, purchase investment plans earning daily ROI, refer others to earn 10% direct bonuses and binary matching income, manage three separate wallets, track earnings and team growth, and withdraw funds. Admins can manage users, approve financial transactions, configure plans and rewards, view comprehensive analytics, and send platform notifications.
